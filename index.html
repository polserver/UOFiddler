About UOFiddler

Welcome to UOFiddler, a powerful tool based on the Ultima SDK for viewing and modifying almost every Ultima Online 2D client file.

UOFiddler has been an essential tool for UO developers and private shard administrators for many years. Whether you're creating custom content, debugging client issues, or building an entirely new world, UOFiddler provides the tools you need.

Source code is released under the Beerware License.

Features

► Art & Gumps
Browse, export, and import static art, items, gumps, and textures with full transparency support.
► Map Editor
View and edit terrain, statics, and map data across all facets of the game world.
► Animations
Preview and modify creature and character animations frame by frame.
► Multi Editor
Design and construct custom multi-tile structures like houses and boats.
► TileData Editor
Edit item and land tile properties including flags, weights, and names.
► Sound & Speech
Browse and export game sounds and speech files.
► Hues
View and edit the color palette used for dyeable items.
► Plugin System
Extend functionality with plugins including UOP Packer, Mass Import, Compare Tool, and more.

Requirements

Version 4.11.0 and newer:

  • .NET Desktop Runtime 8.0.x
  • Windows 10 or newer

For building from source:

  • Visual Studio 2022 v17.10+
  • .NET 8.0 SDK
  • .NET desktop development workload

Download .NET 8.0 at: dotnet.microsoft.com
